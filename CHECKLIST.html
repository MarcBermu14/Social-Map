Checklist de Configuración

Paso 1: Base de Datos
✓

Iniciar MySQL en XAMPP

Abre XAMPP Control Panel y haz click en "Start" para MySQL

✓

Importar sql/schema.sql

Opción A (Recomendado): Abrir phpMyAdmin y pegar el SQL

http://localhost/phpmyadmin → BD citylive → SQL → Ejecutar

Opción B: Línea de comandos

mysql -u root < sql/schema.sql
✓

Verificar BD creada

Busca tabla "users" con campos: id, email, username, verified, verification_token

Abrir phpMyAdmin
Paso 2: Configuración
✓

Crear archivo .env

Copia .env.example a .env en la raíz de citylive/

cp .env.example .env

O copia manualmente en: C:\xampp\htdocs\citylive\.env

✓

Editar .env con credenciales

Abre C:\xampp\htdocs\citylive\.env y configura:

DB_HOST=localhost
DB_NAME=citylive
DB_USER=root
DB_PASS=(tu contraseña MySQL)
SMTP_USER=tu-email@gmail.com
SMTP_PASS=(tu contraseña app)
✓

Verificar configuración

Abre http://localhost/citylive/health-check.php para ver estado

Abrir Health Check
Paso 3: Email SMTP (Opcional)
ℹ️ Nota: Sin configurar SMTP, los emails se simulan en desarrollo. Solo necesitas esto si quieres que se envíen emails de verdad.
✓

Usar Gmail (Recomendado)

1. Crea cuenta en Gmail (si no tienes)

2. Accede a https://myaccount.google.com/security

3. Busca "Contraseñas de aplicación"

4. Selecciona "Mail" y "Windows"

5. Copia la contraseña de 16 caracteres

6. Pega en .env bajo SMTP_PASS

✓

Usar otro SMTP

SendGrid, Mailgun, Office365, etc.

Configura en .env:

SMTP_HOST=smtp.tu-proveedor.com
SMTP_USER=tu@email.com
SMTP_PASS=tu-contraseña
Paso 4: Pruebas
✓

Iniciar Apache

XAMPP Control Panel → Click en "Start" para Apache

✓

Acceder a la aplicación

Abre http://localhost/citylive en tu navegador

Abrir CityLive
✓

Crear cuenta de prueba

Ir a http://localhost/citylive/register.php y llenar formulario

Registrarse
✓

Confirmar email

Si SMTP está configurado: revisar bandeja de entrada

Si NO: revisar logs en C:\xampp\apache\logs\error.log

Ver token en el log y acceder a:
http://localhost/citylive/verify-email.php?token=XXXX
✓

Hacer Login

Una vez verificado el email, ir a http://localhost/citylive/index.php

Ingresa email + contraseña

Login
